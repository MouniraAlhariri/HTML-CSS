## Portfolio Website
- A colorful HTML + CSS portfolio for Mounira.
- Features:
  - Fixed navbar with hover effects
  - Centered profile image with CSS filters and rounded edges
  - Styled table with borders, padding, and colored header
  - Text overlay on images
  - Two-column layout with custom styling
  - Paragraph hover effects
  - Image overlay fade effect
  - Embedded autoplay looping video with text overlay
  - Footer with centered bold text
